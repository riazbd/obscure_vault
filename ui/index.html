Installation Guide
1

Install Python 3.10+

Download from python.org/downloads (Windows/Mac) or use your package manager:

# Ubuntu / Debian / WSL
sudo apt update && sudo apt install python3 python3-pip -y
2

Install FFmpeg

Required for video assembly. Must be available in your system PATH.

# Ubuntu / Debian / WSL
sudo apt install ffmpeg -y

# Mac (Homebrew)
brew install ffmpeg

# Windows — download from ffmpeg.org/download.html
# Then add the /bin folder to your System PATH
3

Install Python Packages

Click the button above, or run manually:

pip install edge-tts moviepy requests Pillow flask
4

Get Free Pexels API Key

Go to pexels.com/api → Sign up (free) → copy your key → paste in Settings tab.

The Pexels API is completely free with no daily limits for standard use.

5

Add Background Music

Go to the Music Library tab and upload dark ambient MP3s. Free sources:

• pixabay.com/music — search "dark ambient", "mystery documentary"
• freemusicarchive.org — filter by CC0 license

6

Generate Your First Video

Go to the Generate Video tab, paste your title and script from Claude, and hit Generate.

How to Start This App

Every time you want to use the pipeline, open a terminal in the obscura_vault folder and run:

python server.py

This opens the UI at http://localhost:5050 in your browser automatically. Keep the terminal open while using the app.

Smart B-roll (semantic per-chunk footage)

          

            Splits the script into ~10 s chunks, asks the LLM for cinematographic
            search queries per chunk, and picks one matching clip per chunk from
            Pexels (and Pixabay if configured). Falls back to the legacy
            substring-keyword search if anything goes wrong.
          

          

            
          

          

            
            
            
Shorter chunks = more cuts = more visual variety, but more downloads.

          

          

            
            
            
Extra free source. Pexels alone usually works fine.
